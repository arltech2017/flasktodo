Create a memory resident database with some known items in the todo list:

  >>> import sqlite3
  >>> conn = sqlite3.connect(':memory:') 
  >>> conn.execute('''CREATE TABLE items (id INTEGER PRIMARY KEY AUTOINCREMENT,
  ...    title TEXT NOT NULL, date TEXT NOT NULL, decription TEXT,
  ...    done BOOLEAN)''')
  <sqlite3.Cursor object at 0x...>
  >>> conn.execute('''INSERT INTO "items" VALUES(1,'Buy groceries',
  ...    '2017-03-15 11:51','Milk, Cheese, Pizza, Fruit, Tylenol',0)''')
  <sqlite3.Cursor object at 0x...>
  >>> conn.execute('''INSERT INTO "items" VALUES(2,'Learn SQL',
  ...    '2017-03-15 11:54','Complete Khan Academy SQL tutorial.',0)''')
  <sqlite3.Cursor object at 0x...>
  >>> conn.commit
  <built-in method commit of sqlite3.Connection object at 0x...>
  >>> cur = conn.execute('SELECT * FROM items')
  >>> items = cur.fetchall() 
  >>> conn.close()

With two items inserted into the database and read back out again, we have
an items list that contains a tuple for each database row containing the
column values as elements. These now need to be converted to a Python
dictionary.

  >>> from tasks import get_tasks
  >>> tasklist = get_tasks(items)
  >>> len(tasklist)
  2
  >>> sorted(tasklist[0])
  ['date', 'description', 'done', 'id', 'title']
